# Plugin-Manifest

Diese Seite ist eine kleine deutsche Prototyp-Übersetzung für ein
lokalisierbares Plugin-Handbuch.

## Was hier gezeigt wird

- Das Plugin bringt eigene Markdown-Dateien mit.
- Das Plugin bringt eine eigene Navigationsdatei mit.
- OpenClaw kann diese Dateien in einen generierten Mintlify-Arbeitsbereich
  kopieren, ohne die quellverwalteten Docs zu überschreiben.

## Noch nicht Teil dieses Schritts

Die Control-UI-Laufzeitübersetzungen werden durch dieses Plugin noch nicht in
Die Laufzeit injiziert. Dieser erste Schritt validiert nur den Docs-Pfad.
